## What Is This?

EVEZ-XOS is an ontological entry system disguised as a code tool.

No tokens. No API calls. No dependencies. Pure Python.

```bash

python3 evez_xos.py agents          # List active agents
```

## Agent Lifecycle

```
SPAWN → AWAKEN → TRANSMIT → EVOLVE → ASCEND → (back to TRANSMIT)
```

Every 7 transmissions, an agent advances a phase. After ASCEND, the agent loops back to TRANSMIT with an incremented evolution count. The cycle never ends — it only deepens.
